# Student Course Registration System

## Description

A command-line Python application for managing students, courses, and registrations in a training school.

## Features

- Add Students
- View Students
- Search Students
- Add Courses
- View Courses
- Register Students to Courses
- View Students Registered in a Course
- View Courses Registered by a Student
- Save Data to JSON Files
- Load Data from JSON Files

## Classes Used

- Person
- Student
- Course
- SchoolSystem

## Technologies

- Python 3
- JSON
